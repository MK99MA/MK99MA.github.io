Last Update: July 21, 2020 - v1.2.5

SoccerMod Edit-19

is a Sourcemod plugin for CS:S.
It features everything needed to play competitive Soccer matches, as well as a few optional but useful additions.

    Main Features:
  • ► Matches - Play competitive or friendly matches with variable length settings.
  • ► Caps - Let 2 captains fight each other and then pick their players in turns.
  • ► Training - Spawn an additional ball or set-up a cannon to train specific situations.
  • ► Referee - Punish players not following the rules or control the scoreboard.
  • ► Sprint - Use a well-timed sprint to get the better over your opponent.
  • ► QoL - Various additions like duckjumpblock, skin-management, admin-management and more...

Last Update: May 05, 2020 - v1.5.0

Shout-Plugin

is a Sourcemod plugin for CS:S.
It can be used to play 'Shouts' during gameplay and tries to recreate a feature of the original Eventscripts Soccermod.

    Main Features:
  • ► Management - Find sounds on the server and add them ingame to the shoutlist or edit its name / path.
  • ► Modes - Different playmodes: Team only, 'Ambiend', Everyone, Radius-based
  • ► Fine-Tuning - Per Shout pitch and volume as well as overall settings.
  • ► Cooldown - Prevent spam with an ingame changeable cooldown.
  • ► Commands - Play the shouts from a menulist or (if toggled) by using a specific command for each sound.

Last Update: August 05, 2020 - v1.2.0

SoccerMod Edit-19-SoScore Addition

is a Sourcemod plugin for CS:S.
A collection of 'modules' from the main soccermod plugin put into a separate plugin to be used on public servers running SoScore by Forlix.

    Main Features:
  • ► Skins - Skins module taken from SoMoE-19 (includes GK-skins).
  • ► DeadChat - DeadChat options included.
  • ► Referee - Punish players not following the rules.
  • ► Sprint - Use a well-timed sprint to get the better over your opponent.
  • ► QoL - Quality of life additions like turned off damage sounds or a death-effect.

Maybe more to come..
