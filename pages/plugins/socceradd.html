SoccerMod Edit-19-SoScore Addition

Last Update: August 05, 2020 Version 1.2.0
Navigation
Description Credits Features Commands Download Installation Changelog

Description

Soccer Mod Edit-19-SoScore Addition is a 'light' slimmed down plugin based on SoMoE-19 features. It is supposed to be a complimentary plugin to Forlix SoScore & SoStats which are better suited for public Soccer Servers. Just like SoMoE-19 it is only aimed at Counter-Strike: Source servers for now.

Credits

Since it is based on SoMoE-19, most of the code was only partially edited by me to suit my needs and fit into one plugin.

Therefore almost all credit should go to the following creators of the respective original works: Original Soccermod - by Marco Boogers - Link Allchat (aka DeadChat) - by Frenzzy - Link ShortSprint - by walmar - Link Advertisements - by DJ Tsunami - Link Updater - by GoD-Tony - Link

Features

Modules:

Sprint: The same configurable sprint as in SoMoE-19. (Different notification modes; changeable length, cooldown etc.)

Adverts: Easily setup adverts to displayed in chat, the center of the screen or any other valid text location. Use different colors, change the interval between messages, etc.

Chat: Colored plugin-related chat messages. Also includes DeadChat to allow spectators / dead people to communicate with the living players.

Skins: Control the used skins (field & gk) as well as handling their download to the clients.

Rules: A separate menu for you to write down your rules, so your players can check them easily at any time.

Menu: Basic menu to control certain settings of the plugin.

Updater: Automatically get any updates and fixes due to the inplementation of updater-support. (If you correctly installed Updater itself!)

Commands

Main Command:

!menu / sm_menu - Opens the main menu.
!admins / sm_admins - Opens the Soccermod online adminlist menu.
!rules / sm_rules - Opens the rules menu.
!gk / sm_gk - Toggle the GK skin.
!help / sm_help - Opens the help menu.
!maprr / sm_maprr - Quickly reload the map.
!reloadads / sm_reloadads - Reload the advertisements.

Installation

I. Download the prerequisites:

To run Soccermod on your server you require the following plugins and files:
Please make sure to choose the right versions (Linux / Windows) depending on the OS of your server!
1. Metamod:Source 1.10 or higher 2. Sourcemod 1.10 or higher OPTIONAL BUT RECOMMENDED 3. Updater.smx and its prerequisites: Steamworks, Official Thread

II. Download Soccermod plugins:

Next download SoScore and its complimentary plugins:
1. SoScore 2. SoStats 3. Soccer_modules.smx (v1.2.0)
 4. Previous versions
A list of previous versions in case you have to roll-back for whatever reason (usually you shouldn't!)

III. Additional Downloads (Optional):

If you don't have your own skins ready, you can download Soccerthemed models & skins here:
1. Default skins
which were already part of the original eventscripts version. 2. Alternate skins
A list of alternate skins people created. If you are using alternate skins make sure to edit soccer_mod_skins.cfg AND soccer_mod_downloads.cfg in your cfg/sm_soccermod/ folder!! Follow this and this for example files. 3. AFK-Kicker
A customizable AFK-Kicker plugin. 3. Duckjumpblock by Marco Boogers
The duckjumpblock plugin which is also part of SoMoE-19.

IV. Extract Files:

After downloading the above files you have to extract its contents. Right click on each zip file and select 'Extract Here'. After extracting the zip files you should have the following folders wherever you extracted the zip files:
addons cfg materials (if you downloaded one of the skin sets) models (if you downloaded one of the skin sets)

V. Copy or Upload the files:

Copy or upload the folders to your server's 'cstrike' folder, for example:
D:\Servers\Counter-Strike Source\cstrike (local server) /home/cstrike (hosted server)

VI. Restart your server (If you freshly installed Sourcemod only!):

Finally, you just have to restart your server. After the restart Soccermod should be fully installed and loaded automatically.
If the plugin isn't loaded, first make sure that you are using the correct versions of Sourcemod and Metamod and the order structure is ok.
