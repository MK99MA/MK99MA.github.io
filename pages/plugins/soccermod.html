SoccerMod Edit-19

Last Update: July 21, 2020 Version 1.2.5
Navigation
Description Credits Features Commands Download Installation Changelog

Description

Soccer Mod Edit-19 is an edited Version of Marco Boogers Soccermod Sourcemod plugin. The edited version is, unlike the original version only compatible with Counter-Strike: Source, but tries to fix minor errors as well as adding new features. A list of all its features and a short description cann be found below. While it is not necessary to read all the information below, doing so should give you a good overview of what you can expect from this plugin and what not. Also, please note that the base code was written by Marco Boogers and this is the first Sourcemod plugin I ever worked on, so the code itself might be inconsistent and heavily improvable. If you do find any issues feel free to create an issue on the plugins Github repo.

Credits

As stated above most of the code was created by Marco Boogers for his original version of the plugin. Next to his base code, I also used code-snippets from other plugins and merely modified those in a way to fit them into one plugin (file).

Therefore almost all credit should go to the following creators of the respective original works: Original Soccermod - by Marco Boogers - Link DuckjumpBlock Script - by Marco Boogers Allchat (aka DeadChat) - by Frenzzy - Link ShortSprint - by walmar - Link Simple AFK kicker - by shavit - Link Updater - by GoD-Tony - Link

Features

General:

Soccermod Menu: A menu to quickly access non-passive features of the plugin. Extended menu as well as changed its command (!menu). Also features a helpmenu giving you a list of every usable command. *MODIFIED*

Commands: Edited and additional commands to quickly access certain aspects of the plugin. *MODIFIED*

Configs: Centralized folder for any plugin related configs (cfg/sm_soccermod). Different configs to control certain aspects of the plugin (main, admins, maps, cap-positions, downloads, gk-areas, skins) *NEW*

Permissions: Every Sourcemod Admin (flag 'b') is automatically granted access to the admin menu. Since not everyone who should have access to the commands (all the time) should also be a Sourcemod admin the plugin features its own small modular admin system. *NEW*

Access-Modes: Different access-modes to limit commands and menu-access. *NEW*

Updater: Automatically get any updates and fixes due to the inplementation of updater-support. (If you correctly installed Updater itself!) *NEW*

Scoreboard: Display the number of goals (kills) and assists (deaths) on the scoreboard, as well as the matchscore. *MODIFIED*

Sprint: Variable sprint settings for server owners, as well as variable notification settings (Chat, Defuse Bar, Hud Timer, Sound) for your players. Usable by binding the command (sm_sprint / say !sprint) or using your '+use' key (Default 'e'). *MODIFIED*

Respawn: Variable respawn for joining players or those killing themselves.

Godmode: Prevents players from being killed by getting hit by the ball.

Skins & Downloads: Control which files should be downloaded to the players and choose which skins (field + gk) should be used.

Statistics: Track points, goals and other stats during a match.

Match:

Starting a match: Start a match with the current match settings either by menu or command (!start). *MODIFIED*

Stopping a match: Immediately stop a running match either by menu or command (!stop). *MODIFIED*

Pausing a match: Pause a running match either by menu or command (!pause / !p). *MODIFIED*

Unpausing a match: Different unpause modes (Fully Manual, ReadyCheck Auto / Manual ). If set to Fully Manual any admin can unpause the match by menu or command (!unpause / !unp). If set to either of the ReadyCheck modes, a menu will pop-up for every player in either CT or T, allowing them to choose their state (Ready / Not Ready). If everyone is ready (or an admin uses the forceunpause command)the match will either unpause automatically or will unlock the unpause command for the admins. *MODIFIED*

Pause: Keep track of the pause duration with a timer and chat message (after the match is unpaused). Unpausing will also start a 5 second countdown (with sound) to prepare player for the resuming match. *MODIFIED*

Match Settings: Control the settings of the match (halftime length, periods, break length) with the ingame Match Menu.

Team Names: Change the names of the two teams (Requires Sourcemod admin permissions). Either use the default values (T/CT), type in a custom name, or use the tag of the teams members as a preset. *NEW*

Match Info: Toggleable and variable display of information regarding the match (Controllable with the Match Menu) are shown in chat when a match starts. In addition most values of relevance are shown in the Match Menu itself. *NEW*

Matchlog: Toggle and view logs of matches ingame. Also creates a physical log in the centralized config folder for belated access. *NEW*

Forfeit Vote: Toggle and control the requirements for a forfeit vote during a match. A successful forfeit vote will immediately abort the ongoing match. *NEW*

Cap:

Spec all: Put all CT and T players to spec in preparation of a cap fight.

Add Random Player: Put a random player from spec into one of the teams.

Cap Fight: Disable Godmode & Sprint and let two captains face each other in a knife-fight for the right of the first pick. *MODIFIED*

Pick Menu: After a Cap Fight is started a menu will be displayed to everyone in spec allowing them to pick any position they can / want to play in the upcoming cap. Any selected position will be shown to the caps in their pick menu. *MODIFIED*

Lock Server: Toggleable and variable setting to automatically lock the server (change the password to a random value) if a Cap is started as well as automatically disabling the state as soon as a Match is started as a follow-up to a cap. While the server is 'locked' an AFK Kick is active, asking inactive players to solve a captcha menu. Failing to do so (wrong or no choice) will kick the player. If the number of players on the server is lower than the pre-set threshold the server will then automatically use the default password until said threshold is reached again. *NEW*

Training:

Cannon: Setup a cannon shooting the ball (Power, Position, Direction, Randomness, Delay between shots). Use it to train your GK reflexes or to simulate other players passes. A bug where people standing on the ball after turning the cannon off would catapult them into air was fixed in this version. *MODIFIED*

Personal Cannon: Every player with the according permissions can also create a personal cannon (which will use the training ball and its own settings for Power, Randomness, etc). *NEW*

Spawn Ball: Every player with the permission to do so can spawn a training ball wherever they're pointing. Spawning and De-Spawning got a small cooldown, and you have to be in one of the two teams to use this feature. *MODIFIED*

Enable/Disable Goals: Disable the goals if you're doing a training session or people are constantly scoring while you are preparating for a match or cap.

Referee:

Yellow Card: Give players a yellow card to warn them for their misbehaviour. A second yellow card will result in a yellow-red card which will put the player to spectator and will also prevent him from joining any team until the card is removed.

Red Card: Give players a red card to punish them for heavy misbehaviour. This will send them directly to spectator and prevent them from joining any of the teams until the card is removed.

Remove Cards: Remove a single yellow or red card, or wipe any currently given cards at once. *MODIFIED*

Control Score: Add or Remove goals for either team to correct the current score (EXAMPLE: A team scores while one or more opponent was afk and they were not ready). You can also reset the score to its initial value of 0:0.

Misc:

Spec Player: Quickly put a player in one of the teams to spectator.

Change maps: Any map added to the plugins maplist will be choosable from a menu for a quick mapchange.

Manage Admins: (Requires rcon permissions in Sourcemod) Add, edit or remove admins ingame. You can modify Soccermod admins, but also Sourcemod admins with this menu. Also features a menu list of admins divided into the two different types (Sourcmeod / Soccer). *NEW*

Allowed maps: Add or remove maps which should utilize the plugin and be added to the Change map submenu.

Ready Check: Toggle and control the settings of the ReadyCheck used during pauses in a match. *NEW*

Damage Sounds: Disable the damage sound players would hear when they are hit by the ball. *NEW*

Remove Ragdolls: Choose 1 out of 3 modes to handle the ragdolls of dying players. No - do nothing; Yes - despawn them; Dissolve - Apply the dissolve effect from HL2 and remove them. *NEW*

DuckjumpBlock: Tries to revert the duckjump "state" if a player tries to gain an advantage by using it in an airduel. *NEW*

Shout Plugin: If my Shout-Plugin is installed on the server shortcuts for its 2 commands are added to the Soccermod Menu. *NEW*

Chat:

Chat Style: Control the chat style of any plugin related messages. You can choose the prefix, the chat color as well as the prefix color. *MODIFIED*

MVP Messages: Since the original plugin comes with a point system, you can display the MVP of the round in chat and scoreboard or disable it. *MODIFIED*

DeadChat: Toggle and control deadchat settings. *NEW*

Commands

Main Command:

!menu / sm_menu - Opens the Soccermod main menu.
!madmin / sm_madmin - Opens the Soccermod admin menu. *NEW*
!gk / sm_gk - Toggle the GK skin. *MODIFIED*
!maprr / sm_maprr - Quickly reload the map. *NEW*
!rr / sm_rr - Quickly restart the round. *MODIFIED*

Menu Shortcut Commands:

!match / sm_match - Opens the Soccermod match menu. *MODIFIED*
!cap / sm_cap - Opens the Soccermod cap menu. *MODIFIED*
!training / sm_training - Opens the Soccermod training menu. *NEW*
!ref / sm_ref - Opens the Soccermod referee menu. *NEW*
!soccerset / sm_soccerset - Opens the Soccermod settings menu. *NEW*

Info Menu Commands:

!commands / sm_commands - Opens the Soccermod commands menu. *MODIFIED*
!help / sm_help - Opens the Soccermod help menu. *MODIFIED*
!admins / sm_admins - Opens the Soccermod online adminlist menu. *NEW*
!stats / sm_stats - Opens the Soccermod stats menu. *MODIFIED*
!rank / sm_rank - Opens the Soccermod rank menu. *MODIFIED*
!credits / sm_credits - Opens the Soccermod credits menu. *MODIFIED*

Match Commands:

!start / sm_start - Starts a match. *NEW*
!pause; !p / sm_pause; sm_p - Pauses a match. *NEW*
!unpause; !unp / sm_unpause; sm_unp - Unpauses a match. *NEW*
!stop / sm_stop - Stops a match. *NEW*
!matchrr / sm_matchrr - Quickly restart the current match. *NEW*
!rdy / sm_rdy - (Re-)Opens the ReadyCheck Menu. *NEW*

Cap Commands:

!pick / sm_pick - (Re-)Opens the Soccermod pick menu. *MODIFIED*

Admin Commands: (Requires SM RCON flag)

!addadmin / sm_addadmin - Add an admin to the Sourcemod admins_simple.ini file. *NEW*
!dpass / sm_dpass - Reset the server password to the default value. *NEW*
!forcerdy / sm_forcerdy - Forces every player to be set to 'ready' during the current pause. *NEW*
!forceunp / sm_forceunp - Forces the match to unpause. *NEW*
!pass / sm_pass - Set a custom server password. *NEW*
!rpass / sm_rpass - Set a random server password. *NEW*

Installation

I. Download the prerequisites:

To run Soccermod on your server you require the following plugins and files:
Please make sure to choose the right versions (Linux / Windows) depending on the OS of your server!
1. Metamod:Source 1.10 or higher 2. Sourcemod 1.10 or higher OPTIONAL BUT RECOMMENDED 3. Updater.smx and its prerequisites: Steamworks, Official Thread

II. Download Soccermod:

Next download Soccermod itself. There are 3 possible options to choose from:
1. Full Package (v1.2.5)
Includes the plugin, source and the training ball model. 2. Soccermod.smx (v1.2.5)
Includes only the latest plugin file for manual updates. 3. Previous versions
A list of previous versions in case you have to roll-back for whatever reason (usually you shouldn't!)

III. Additional Downloads (Optional):

If you don't have your own skins ready, you can download Soccerthemed models & skins here:
1. Default skins
which were already part of the original eventscripts version. 2. Alternate skins
A list of alternate skins people created. If you are using alternate skins make sure to edit soccer_mod_skins.cfg AND soccer_mod_downloads.cfg in your cfg/sm_soccermod/ folder!! Follow this and this for example files.

IV. Extract Files:

After downloading the above files you have to extract its contents. Right click on each zip file and select 'Extract Here'. After extracting the zip files you should have the following folders wherever you extracted the zip files:
addons cfg materials models sounds

V. Copy or Upload the files:

Copy upload the folders to your server's 'cstrike' folder, for example:
D:\Servers\Counter-Strike Source\cstrike (local server) /home/cstrike (hosted server)

VI. Restart your server:

Finally, you just have to restart your server. After the restart Soccermod should be fully installed and loaded automatically.
If the plugin isn't loaded, first make sure that you are using the correct versions of Sourcemod and Metamod and the order structure is ok.
