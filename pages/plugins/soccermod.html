SoccerMod Edit-19

Last Update: July 21, 2020 Version 1.2.5
Navigation
Description Credits Features Installation Download Changelog

Description

Soccer Mod Edit-19 is an edited Version of Marco Boogers Soccermod Sourcemod plugin. The edited version is, unlike the original version only compatible with Counter-Strike: Source, but tries to fix minor errors as well as adding new features. A list of all its features and a short description cann be found below. While it is not necessary to read all the information below, doing so should give you a good overview of what you can expect from this plugin and what not. Also, please note that the base code was written by Marco Boogers and this is the first Sourcemod plugin I ever worked on, so the code itself might be inconsistent and heavily improvable. If you do find any issues feel free to create an issue on the plugins Github repo.

Credits

As stated above most of the code was created by Marco Boogers for his original version of the plugin. Next to his base code, I also used code-snippets from other plugins and merely modified those in a way to fit them into one plugin (file).

Features

General:

Soccermod Menu: A menu to quickly access non-passive features of the plugin. Extended menu as well as changed its command (!menu). Also features a helpmenu giving you a list of every usable command. *MODIFIED*

Commands: Edited and additional commands to quickly access certain aspects of the plugin. *MODIFIED*

Configs: Centralized folder for any plugin related configs (cfg/sm_soccermod). Different configs to control certain aspects of the plugin (main, admins, maps, cap-positions, downloads, gk-areas, skins) *NEW*

Permissions: Every Sourcemod Admin (flag 'b') is automatically granted access to the admin menu. Since not everyone who should have access to the commands (all the time) should also be a Sourcemod admin the plugin features its own small modular admin system. *NEW*

Access-Modes: Different access-modes to limit commands and menu-access. *NEW*

Updater: Automatically get any updates and fixes due to the inplementation of updater-support. (If you correctly installed Updater itself!) *NEW*

Scoreboard: Display the number of goals (kills) and assists (deaths) on the scoreboard, as well as the matchscore. *MODIFIED*

Sprint: Variable sprint settings for server owners, as well as variable notification settings (Chat, Defuse Bar, Hud Timer, Sound) for your players. Usable by binding the command (sm_sprint / say !sprint) or using your '+use' key (Default 'e'). *MODIFIED*

Respawn: Variable respawn for joining players or those killing themselves.

Godmode: Prevents players from being killed by getting hit by the ball.

Skins & Downloads: Control which files should be downloaded to the players and choose which skins (field + gk) should be used.

Statistics: Track points, goals and other stats during a match.

Match:

Starting a match: Start a match with the current match settings either by menu or command (!start). *MODIFIED*

Stopping a match: Immediately stop a running match either by menu or command (!stop). *MODIFIED*

Pausing a match: Pause a running match either by menu or command (!pause / !p). *MODIFIED*

Unpausing a match: Different unpause modes (Fully Manual, ReadyCheck Auto / Manual ). If set to Fully Manual any admin can unpause the match by menu or command (!unpause / !unp). If set to either of the ReadyCheck modes, a menu will pop-up for every player in either CT or T, allowing them to choose their state (Ready / Not Ready). If everyone is ready (or an admin uses the forceunpause command)the match will either unpause automatically or will unlock the unpause command for the admins. *MODIFIED*

Pause: Keep track of the pause duration with a timer and chat message (after the match is unpaused). Unpausing will also start a 5 second countdown (with sound) to prepare player for the resuming match. *MODIFIED*

Match Settings: Control the settings of the match (halftime length, periods, break length) with the ingame Match Menu.

Team Names: Change the names of the two teams (Requires Sourcemod admin permissions). Either use the default values (T/CT), type in a custom name, or use the tag of the teams members as a preset. *NEW*

Match Info: Toggleable and variable display of information regarding the match (Controllable with the Match Menu) are shown in chat when a match starts. In addition most values of relevance are shown in the Match Menu itself. *NEW*

Matchlog: Toggle and view logs of matches ingame. Also creates a physical log in the centralized config folder for belated access. *NEW*

Cap:

Spec all: Put all CT and T players to spec in preparation of a cap fight.

Add Random Player: Put a random player from spec into one of the teams.

Cap Fight: Disable Godmode & Sprint and let two captains face each other in a knife-fight for the right of the first pick. *MODIFIED*

Pick Menu: After a Cap Fight is started a menu will be displayed to everyone in spec allowing them to pick any position they can / want to play in the upcoming cap. Any selected position will be shown to the caps in their pick menu. *MODIFIED*

Lock Server: Toggleable and variable setting to automatically lock the server (change the password to a random value) if a Cap is started as well as automatically disabling the state as soon as a Match is started as a follow-up to a cap. While the server is 'locked' an AFK Kick is active, asking inactive players to solve a captcha menu. Failing to do so (wrong or no choice) will kick the player. If the number of players on the server is lower than the pre-set threshold the server will then automatically use the default password until said threshold is reached again. *NEW*

Training:

Soccermod Menu: A menu to quickly access non-passive features of the plugin. Extended menu as well as changed its command (!menu). *MODIFIED*

Referee:

Soccermod Menu: A menu to quickly access non-passive features of the plugin. Extended menu as well as changed its command (!menu). *MODIFIED*

Misc:

Soccermod Menu: A menu to quickly access non-passive features of the plugin. Extended menu as well as changed its command (!menu). *MODIFIED*

Chat:

Soccermod Menu: A menu to quickly access non-passive features of the plugin. Extended menu as well as changed its command (!menu). *MODIFIED*
