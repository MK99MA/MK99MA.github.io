Description

This is a small list of existing Counter-Strike: Source Soccermaps. Some of these may be created by me, but I'll also list maps created by other people and credit them. If I listed your map and you don't want me to include it on this page for whatever reason please contact me directly about the issue.

6v6

ka_soccer_xsl_stadium_b1:

Probably one of the best known CS:S Soccermaps created by Marco Boogers. It was the official map for the XSL and other following leagues and is still used on some servers.

ka_soccer_avalon_v8:

A custom map for the ISL team Avalon created by L!ncoln. The ball uses a combination of a func_physbox and a model. It also features a working scoreboard (with control buttons).

ka_soccer_comp_2020_final_v3:

A reworked map featuring a new stadium design as well as new textures created by Natsu. The ball is 'identical' to the one on xsl_stadium, but still shows some differeces in behaviour.

Indoor

ka_indoor_avalon_v2:

An indoor map suitable for 3v3 up to 5v5 matches, created by L!ncoln. As the name suggests it was also created for the team Avalon and uses the same assets as the 6v6 map. It also features two hidden buttons to swap between normal and round corners.

ka_indoor_avalon_small_v2:

An indoor map suitable for 2v2 or 3v3 matches, created by L!ncoln. As the name suggests it was also created for the team Avalon and uses the same assets as the 6v6 map. It also features two hidden buttons to swap between normal and round corners.

Misc

ka_parkhead:

A bigger map trying to recreate the real stadium 'Celtic Park', created by ???. Because of it's bigger size it is suitable for 8v8 matches.

ka_skillrun_avalon_1vs1_v4:

A skillrun or training map, created by L!ncoln. It consists of 5 soccer-related stages, featuring a goal-wall shootout and other target practices.
