Description

This is a small list of existing Counter-Strike: Global Offensive Soccermaps. Some of these may be created by me, but I'll also list maps created by other people and credit them. If I listed your map and you don't want me to include it on this page for whatever reason please contact me directly about the issue.

6v6

ka_soccer_stadium_skbx:

A normal sized map for CS:GO created by MK99. It features a 'secret room' with different buttons to toggle environment modes. Those modes will change the sky texture, add floodlights or even weather effects.

Indoor

ka_soccer_multiindoor_a:

An indoor map for CS:GO created by MK99. It features different fieldsizes that can be switched at any time. To enter the room for switching the modes, a button will be accessible the first 15 seconds of every round. A second version that features a working radar (but will have other minor issues) is also available.

ka_soccer_multiindoor_b:

An indoor map for CS:GO created by MK99. It features different fieldsizes that can be switched at any time. To enter the room for switching the modes, a button will be accessible the first 15 seconds of every round. A second version without a working radar but also less issues is also available.
